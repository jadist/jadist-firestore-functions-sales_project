# Readme

## Description
This project is a backend-type project of Jadist. Created to be run on Firebase Serverless Function. And only be used for Jadist of Sales project only. That consist of, but not limited to:
- Quotation
- Purchase Order
- Sales Invoice
- Sales Order
- Online Payment
- Delivery Order
- Cash Collection
- Received Note
- Cash Registration
- Closing
